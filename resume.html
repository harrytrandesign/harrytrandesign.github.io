Find out what I've been doing

My journey began with a print("Hello World") take a look at my progress since

  • Hello World Mid 2013
    The very first time I printed hello world into a console. I signed up for some courses through Team Treehouse and started exploring the various subjects that were available.
  • My First Client Early 2014
    I started creating websites for clients with the knowledge that I learned from various tutorials. One of my first clients I helped create a website for was a restaurant located in Iowa called Pho, most of the work on this project was done remotely from my office in Sacramento, CA.
  • Mobile Game Mid 2014
    Installed Corona SDK and started working on mobile game development projects. At the time I was studying Lua and working on mobile game tutorials using the Corona SDK IDE. Later, I was able to load my projects onto my mobile devices for testing.
  • iOS + Xcode 2015
    Took a course on Udemy covering iOS development with Swift and Xcode. Learned the basics for creating views and methods, then decided to try out Android development because I was a part of the Android ecosystem.
  • Java + Android Studio 2016
    After taking a course on Android development on Udemy and Team Treehouse, I created several practice apps. I set out to create my own application that would be useful for myself, this application turned into "Short Notes" an application which keeps very short notes directly on the users device. Other applications I developed at this time included a shower timer which reminds us to conserve water, and a tip calculator.
  • Firebase BaaS + The Cloud 2017
    Learned Firebase BaaS including authentication, real time database, and cloud storage and implemented these new features into an application I created that allowed users to submit funny photos of their pets to share with others.
